## 🚀 Run Python
- MacOS
```
python3 <file_name>
```
- Window
```
python or py or pip
```
- check version
```
python --version
```

## 🖥️ Command Line
- list file in directory (ภาพรวม)
```
ls
```
- change directory (เข้าไป)
```
cd <folder_name>
```
- create directory (สร้าง folder)
```
mkdir <folder_name>
```
- create file (สร้าง file)
```
touch <file_name>
```
- delete folder (ลบ folder)
```
rm -rf <folder_name>
```
